### 🎯 주요 메시지
- **마케팅 예산 50% 절감**

## 🛠 기술 스택
- **HTML5** - 시맨틱 마크업
- **CSS3** - 반응형 디자인
- **Vanilla JavaScript** - 인터랙티브 기능
- **Tailwind CSS** - 빠른 스타일링
